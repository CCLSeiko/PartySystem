# DonationSystem 使用者故事

## 捐款者（一般使用者）

### 1. 註冊與登入

**故事 1：作為捐款者，我希望可以註冊帳號，以便使用系統功能。**
- 接受標準：
  - 可以使用電子郵件註冊
  - 系統發送驗證郵件
  - 驗證後可登入系統

**故事 2：作為捐款者，我希望可以使用電子郵件與密碼登入，以便管理我的捐款。**
- 接受標準：
  - 輸入正確的電子郵件與密碼後可登入
  - 登入失敗時顯示錯誤訊息
  - 支援記住我功能

**故事 3：作為捐款者，我希望可以查看與編輯個人資料，以便保持資訊正確。**
- 接受標準：
  - 可以查看個人資料（姓名、電話、地址等）
  - 可以編輯個人資料
  - 可以填寫身分證字號（選填）
  - 可以設定是否同意捐贈資料上傳國稅局
  - 變更後即時更新

### 2. 捐款功能

**故事 4：作為捐款者，我希望可以選擇捐款方式（現金、信用卡、郵政劃撥），以便使用最方便的付款方式。**
- 接受標準：
  - 顯示所有可用的捐款方式
  - 點選後進入對應的付款流程
  - 每種方式都有清楚的說明

**故事 5：作為捐款者，我希望可以輸入捐款金額，以便決定捐款數額。**
- 接受標準：
  - 可以輸入任意金額（符合最小金額限制）
  - 系統顯示捐款金額確認
  - 提供常見金額快速選擇（例如 100、500、1000 元）

**故事 6：作為捐款者，我希望可以選擇捐款用途，以便指定捐款用途。**
- 接受標準：
  - 顯示可用的捐款用途選項
  - 可以選擇一般捐款或特定專案捐款
  - 可以輸入備註資訊

**故事 7：作為捐款者，我希望可以確認捐款資訊，以免輸入錯誤。**
- 接受標準：
  - 顯示完整的捐款資訊（金額、方式、用途等）
  - 提供修改按鈕
  - 確認後進入付款流程

**故事 8：作為捐款者，我希望可以查看捐款歷史紀錄，以便追蹤我的捐款。**
- 接受標準：
  - 顯示所有捐款紀錄（日期、金額、方式、狀態）
  - 可以搜尋特定捐款
  - 可以下載捐款收據

**故事 9：作為捐款者，我希望可以在一定時間內取消捐款，以免後悔。**
- 接受標準：
  - 顯示可取消的捐款紀錄
  - 取消後顯示確認訊息
  - 記錄取消原因

### 3. 支付功能

**故事 10：作為捐款者，我希望可以使用信用卡付款，以便快速完成捐款。**
- 接受標準：
  - 可以輸入信用卡資訊
  - 支援 3D Secure 驗證
  - 付款成功後顯示確認訊息

**故事 11：作為捐款者，我希望可以使用郵政劃撥付款，以便使用傳統付款方式。**
- 接受標準：
  - 產生郵政劃撥單
  - 提供劃撥單下載與列印功能
  - 顯示劃撥單處理狀態

**故事 12：作為捐款者，我希望可以使用現金付款，以便現場捐款。**
- 接受標準：
  - 記錄現金捐款資訊
  - 提供收據列印功能
  - 顯示捐款確認訊息

### 4. 定期定額捐款 (新增)

**故事 19：作為捐款者，我希望可以設定每月定期定額捐款，以便持續支持機構。**
- 接受標準：
  - 可以選擇扣款頻率（每月/每季/每年）
  - 可以設定每期金額
  - 可以設定總期數或選擇無限期
  - 首次扣款立即執行
  - 顯示預計扣款日期

**故事 20：作為捐款者，我希望可以查看我的定期定額方案，以便了解捐款狀況。**
- 接受標準：
  - 顯示所有定期定額方案列表
  - 顯示每期金額、頻率、已扣款次數
  - 顯示下次扣款日期
  - 顯示方案狀態（啟用/暫停/已取消）

**故事 21：作為捐款者，我希望可以修改定期定額設定，以便調整捐款金額或頻率。**
- 接受標準：
  - 可以修改每期金額
  - 可以暫停定期定額
  - 可以恢復已暫停的定期定額
  - 修改後顯示確認訊息

**故事 22：作為捐款者，我希望可以取消定期定額捐款，以便停止持續捐款。**
- 接受標準：
  - 取消後顯示確認訊息
  - 記錄取消原因（選填）
  - 取消後不再執行扣款

**故事 23：作為捐款者，我希望在扣款失敗時收到通知，以便更新付款資訊。**
- 接受標準：
  - 扣款失敗時發送 Email 通知
  - 顯示失敗原因
  - 提供更新信用卡資訊的連結

### 5. 匿名捐款 (新增)

**故事 24：作為訪客，我希望可以不註冊帳號直接捐款，以便快速完成捐款。**
- 接受標準：
  - 不需登入即可選擇捐款方式
  - 可選填姓名、Email 等聯絡資訊
  - 捐款完成後顯示確認頁面
  - 系統提供捐款查詢編號

**故事 25：作為匿名捐款者，我希望可以透過 Email 查詢捐款紀錄，以便追蹤我的捐款。**
- 接受標準：
  - 輸入 Email 與捐款編號後可查詢
  - 顯示捐款金額、日期、狀態
  - 可下載捐款收據

## 管理員

### 6. 捐款管理

**故事 13：作為管理員，我希望可以查看所有捐款紀錄，以便管理捐款。**
- 接受標準：
  - 顯示所有捐款紀錄（含匿名捐款）
  - 可以搜尋與篩選捐款
  - 可以匯出捐款資料

**故事 14：作為管理員，我希望可以查看統計報表，以便分析捐款情況。**
- 接受標準：
  - 顯示捐款總額統計
  - 顯示捐款方式分佈
  - 顯示捐款用途分佈
  - 顯示定期定額執行率（新增）

### 7. 對帳管理 (新增)

**故事 26：作為管理員，我希望可以上傳郵局的對帳檔案，以便核對郵政劃撥捐款。**
- 接受標準：
  - 支援上傳 CSV/TXT 格式對帳檔案
  - 系統自動解析並比對捐款紀錄
  - 顯示相符與不相符的筆數
  - 防止重複上傳相同檔案

**故事 27：作為管理員，我希望可以查看對帳結果，以便處理差異。**
- 接受標準：
  - 顯示對帳歷史紀錄
  - 顯示每筆對帳的詳細比對結果
  - 可以下載對帳差異報告
  - 可以手動修正不相符紀錄

### 8. 稅務申報 (新增)

**故事 28：作為管理員，我希望可以匯出年度捐款資料，以便進行國稅局申報。**
- 接受標準：
  - 可選擇申報年度
  - 匯出符合國稅局格式的 CSV 檔案
  - 僅包含已同意資料上傳的捐款人
  - 顯示年度捐款總額與人數統計

### 9. 使用者管理

**故事 15：作為管理員，我希望可以管理使用者帳號，以便維護系統安全。**
- 接受標準：
  - 查看使用者列表
  - 搜尋使用者
  - 停用/啟用使用者帳號

### 10. 系統設定

**故事 16：作為管理員，我希望可以設定系統參數，以便調整系統行為。**
- 接受標準：
  - 設定捐款最小金額
  - 設定捐款用途選項
  - 設定支付閘道參數
  - 設定定期定額重試次數（新增）
  - 設定稅務申報年度（新增）

### 11. 定期定額管理 (新增)

**故事 29：作為管理員，我希望可以查看所有定期定額方案，以便管理定期捐款。**
- 接受標準：
  - 顯示所有使用者的定期定額方案
  - 可以依狀態（啟用/暫停/已取消）篩選
  - 顯示扣款成功率統計
  - 可以手動暫停或啟用特定方案

## 系統管理員

### 12. 系統維護

**故事 17：作為系統管理員，我希望可以監控系統狀態，以便及時處理問題。**
- 接受標準：
  - 顯示系統運行狀態
  - 記錄系統日誌
  - 發送錯誤通知
  - 定期定額扣款失敗告警（新增）
  - 對帳差異過大告警（新增）

**故事 18：作為系統管理員，我希望可以設定系統參數，以便調整系統設定。**
- 接受標準：
  - 設定系統時區
  - 設定備份策略
  - 設定安全性參數
  - 設定 Cloud KMS 加密金鑰輪換週期（新增）
  - 設定背景任務佇列參數（新增）
