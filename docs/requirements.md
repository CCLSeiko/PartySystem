# DonationSystem 需求文件

## 系統概述

DonationSystem 是一個線上捐款平台，支援多種捐款方式（現金、信用卡扣款、郵政劃撥、定期定額扣款），提供使用者便捷的捐款體驗與後台管理功能。

## 目標使用者

1. **捐款者**：一般民眾，希望透過線上平台進行捐款
2. **管理員**：機構管理人員，負責管理捐款紀錄、對帳與統計
3. **系統管理員**：技術人員，負責系統維護與設定

## 功能需求

### 1. 使用者管理

#### 1.1 註冊
- 使用者可透過電子郵件註冊帳號
- 需驗證電子郵件地址
- 密碼需符合安全性要求（至少 8 字元，包含大小寫與數字）
- 可填寫身分證字號/統編（選填，用於稅務申報）

#### 1.2 登入
- 使用者可使用電子郵件與密碼登入
- 支援記住我功能（長期 Session）
- 登入失敗次數限制（防止暴力破解）

#### 1.3 個人資料管理
- 使用者可查看與編輯個人資料（姓名、電話、地址等）
- 使用者可更改密碼
- 使用者可查看捐款歷史紀錄
- 使用者可設定/取消稅務申報同意
- 使用者可管理定期定額捐款方案

### 2. 捐款管理

#### 2.1 建立捐款
- 使用者可選擇捐款方式（現金、信用卡、郵政劃撥）
- 使用者可選擇定期定額捐款方式
- 使用者可輸入捐款金額
- 使用者可選擇捐款用途（一般捐款、專案捐款等）
- 使用者可填寫備註資訊
- 系統需驗證捐款金額（最小金額限制）
- **支援匿名捐款：訪客不需註冊即可捐款，提供選填聯絡資訊**

#### 2.2 查詢捐款
- 使用者可查看自己的捐款紀錄
- 使用者可搜尋特定捐款（依日期、金額、用途等）
- 使用者可下載捐款收據（PDF 格式）
- 匿名捐款人可透過 Email + 捐款編號查詢捐款狀態

#### 2.3 取消捐款
- 使用者可在一定時間內取消捐款（例如 24 小時內）
- 取消捐款需記錄原因

### 3. 定期定額捐款 (核心功能)

#### 3.1 建立定期定額
- 使用者可選擇扣款頻率（每月/每季/每年）
- 使用者可設定每期金額
- 使用者可選擇捐款用途
- 使用者可設定總期數（或無限期）
- 首次扣款立即執行，後續自動排程

#### 3.2 管理定期定額
- 使用者可查看定期定額方案列表
- 使用者可修改金額、頻率等設定
- 使用者可暫停/恢復定期定額
- 使用者可取消定期定額

#### 3.3 自動扣款
- 系統自動執行到期扣款
- 扣款失敗自動重試（最多 3 次）
- 連續失敗通知使用者
- 扣款成功發送收據

### 4. 支付整合

#### 4.1 信用卡扣款
- 整合第三方支付閘道（Stripe 主要 + 藍新金流備援）
- 支援主流信用卡（Visa、MasterCard、JCB 等）
- 支援 3D Secure 驗證
- 支援分期付款（可選）
- 支援定期定額授權 Token 化

#### 4.2 郵政劃撥
- 產生郵政劃撥單（含劃撥帳號、金額、備註）
- 提供劃撥單下載與列印功能
- 記錄劃撥單狀態（已產生、已寄出、已收款）
- **支援對帳檔案匯入與批次核對**

#### 4.3 現金捐款
- 記錄現金捐款資訊（捐款人、金額、時間、地點）
- 提供收據列印功能

#### 4.4 金流 Webhook
- 接收 Stripe/藍新金流異步交易結果通知
- Webhook 簽章驗證
- 自動更新捐款與支付狀態

### 5. 對帳管理 (新增)

#### 5.1 對帳檔案匯入
- 管理員可上傳郵局/金流商提供的對帳檔案（CSV/TXT）
- 系統自動解析檔案內容
- 自動比對捐款紀錄
- 標記相符與不相符的紀錄

#### 5.2 對帳結果查詢
- 查看對帳歷史紀錄
- 下載對帳差異報告
- 支援人工修正不相符紀錄

### 6. 稅務申報 (新增)

#### 6.1 個人資料管理
- 記錄捐款人身分證字號/統一編號（AES-256-GCM 加密）
- 記錄捐款人是否同意資料上傳國稅局

#### 6.2 年度申報
- 匯出符合國稅局格式的 CSV 檔案
- 年度捐款總額統計
- 已同意/未同意申報捐款人清單

### 7. 管理後台

#### 7.1 捐款管理
- 查看所有捐款紀錄
- 搜尋與篩選捐款（依日期、金額、捐款方式、狀態等）
- 匯出捐款資料（CSV、Excel 格式）

#### 7.2 統計報表
- 捐款總額統計（日、週、月、年）
- 捐款方式分佈統計
- 捐款用途分佈統計
- 捐款者統計（新捐款者、回流捐款者）
- 定期定額執行率統計（新增）

#### 7.3 使用者管理
- 查看使用者列表
- 搜尋使用者（依姓名、電子郵件等）
- 停用/啟用使用者帳號

#### 7.4 對帳管理 (新增)
- 上傳對帳檔案
- 查看對帳進度與結果
- 處理對帳差異

#### 7.5 系統設定
- 設定捐款最小金額
- 設定捐款用途選項
- 設定支付閘道參數
- 設定定期定額重試次數（新增）
- 設定稅務申報年度（新增）

## 非功能需求

### 1. 效能需求
- 系統回應時間：頁面載入 < 3 秒，API 回應 < 1 秒
- 支援同時 1000 人使用
- 資料庫查詢效能：95% 查詢 < 500ms
- 定期定額扣款批次處理：每日檢查 < 30 秒

### 2. 安全性需求
- 使用 HTTPS 傳輸資料（TLS 1.3）
- 密碼加密儲存（使用 bcrypt 或 Argon2）
- **敏感個資欄位層級加密（AES-256-GCM）**
- **身分證字號、電話、地址等 PII 在應用層加密**
- **加密金鑰由 Cloud KMS 管理**
- Webhook 端點使用金流商簽章驗證
- 防止 SQL 注入、XSS、CSRF 攻擊
- 定期安全漏洞掃描
- 個資存取稽核日誌

### 3. 可用性需求
- 系統可用性：99.9%（每月 downtime < 43 分鐘）
- 支援多瀏覽器（Chrome、Firefox、Safari、Edge）
- 支援行動裝置（RWD 設計）

### 4. 可擴充性需求
- 模組化設計，易於新增功能
- 支援水平擴充（負載均衡）
- API 設計符合 RESTful 標準
- **非同步處理使用訊息佇列（Cloud Pub/Sub）**

### 5. 法規遵循
- 符合個資法（個人資料保護）
- 符合稅務規定（捐款收據格式、國稅局申報格式）
- 符合支付行業安全標準（PCI DSS Level 1）
- 符合台灣綜合所得稅扣除額電子化申報規範

## 假設與限制

### 假設
1. 使用者擁有有效的電子郵件地址
2. 使用者擁有信用卡或郵政劃撥帳號
3. 網路連線穩定
4. 第三方支付閘道服務正常運作
5. 郵局提供對帳檔案（CSV/TXT 格式）

### 限制
1. 捐款金額上限：單筆 100 萬元（可設定）
2. 捐款取消時間限制：24 小時內
3. 郵政劃撥單處理時間：3-5 個工作天
4. 定期定額僅支援信用卡扣款（初期）
5. 信用卡分期付款僅支援特定銀行
6. 定期定額連續 3 次扣款失敗自動暫停
7. 年度國稅局申報資料僅供上年度使用
