# DonationSystem 使用者故事

## 捐款者（一般使用者）

### 1. 註冊與登入

**故事 1：作為捐款者，我希望可以註冊帳號，以便使用系統功能。**
- 接受標準：
  - 可以使用電子郵件註冊
  - 系統發送驗證郵件
  - 驗證後可登入系統

**故事 2：作為捐款者，我希望可以使用電子郵件與密碼登入，以便管理我的捐款。**
- 接受標準：
  - 輸入正確的電子郵件與密碼後可登入
  - 登入失敗時顯示錯誤訊息
  - 支援記住我功能

**故事 3：作為捐款者，我希望可以查看與編輯個人資料，以便保持資訊正確。**
- 接受標準：
  - 可以查看個人資料（姓名、電話、地址等）
  - 可以編輯個人資料
  - 變更後即時更新

### 2. 捐款功能

**故事 4：作為捐款者，我希望可以選擇捐款方式（現金、信用卡、郵政劃撥），以便使用最方便的付款方式。**
- 接受標準：
  - 顯示所有可用的捐款方式
  - 點選後進入對應的付款流程
  - 每種方式都有清楚的說明

**故事 5：作為捐款者，我希望可以輸入捐款金額，以便決定捐款數額。**
- 接受標準：
  - 可以輸入任意金額（符合最小金額限制）
  - 系統顯示捐款金額確認
  - 提供常見金額快速選擇（例如 100、500、1000 元）

**故事 6：作為捐款者，我希望可以選擇捐款用途，以便指定捐款用途。**
- 接受標準：
  - 顯示可用的捐款用途選項
  - 可以選擇一般捐款或特定專案捐款
  - 可以輸入備註資訊

**故事 7：作為捐款者，我希望可以確認捐款資訊，以免輸入錯誤。**
- 接受標準：
  - 顯示完整的捐款資訊（金額、方式、用途等）
  - 提供修改按鈕
  - 確認後進入付款流程

**故事 8：作為捐款者，我希望可以查看捐款歷史紀錄，以便追蹤我的捐款。**
- 接受標準：
  - 顯示所有捐款紀錄（日期、金額、方式、狀態）
  - 可以搜尋特定捐款
  - 可以下載捐款收據

**故事 9：作為捐款者，我希望可以在一定時間內取消捐款，以免後悔。**
- 接受標準：
  - 顯示可取消的捐款紀錄
  - 取消後顯示確認訊息
  - 記錄取消原因

### 3. 支付功能

**故事 10：作為捐款者，我希望可以使用信用卡付款，以便快速完成捐款。**
- 接受標準：
  - 可以輸入信用卡資訊
  - 支援 3D Secure 驗證
  - 付款成功後顯示確認訊息

**故事 11：作為捐款者，我希望可以使用郵政劃撥付款，以便使用傳統付款方式。**
- 接受標準：
  - 產生郵政劃撥單
  - 提供劃撥單下載與列印功能
  - 顯示劃撥單處理狀態

**故事 12：作為捐款者，我希望可以使用現金付款，以便現場捐款。**
- 接受標準：
  - 記錄現金捐款資訊
  - 提供收據列印功能
  - 顯示捐款確認訊息

## 管理員

### 4. 捐款管理

**故事 13：作為管理員，我希望可以查看所有捐款紀錄，以便管理捐款。**
- 接受標準：
  - 顯示所有捐款紀錄
  - 可以搜尋與篩選捐款
  - 可以匯出捐款資料

**故事 14：作為管理員，我希望可以查看統計報表，以便分析捐款情況。**
- 接受標準：
  - 顯示捐款總額統計
  - 顯示捐款方式分佈
  - 顯示捐款用途分佈

### 5. 使用者管理

**故事 15：作為管理員，我希望可以管理使用者帳號，以便維護系統安全。**
- 接受標準：
  - 查看使用者列表
  - 搜尋使用者
  - 停用/啟用使用者帳號

### 6. 系統設定

**故事 16：作為管理員，我希望可以設定系統參數，以便調整系統行為。**
- 接受標準：
  - 設定捐款最小金額
  - 設定捐款用途選項
  - 設定支付閘道參數

## 系統管理員

### 7. 系統維護

**故事 17：作為系統管理員，我希望可以監控系統狀態，以便及時處理問題。**
- 接受標準：
  - 顯示系統運行狀態
  - 記錄系統日誌
  - 發送錯誤通知

**故事 18：作為系統管理員，我希望可以設定系統參數，以便調整系統設定。**
- 接受標準：
  - 設定系統時區
  - 設定備份策略
  - 設定安全性參數
